Finite presentation basics

2014, May 29

$$ \newcommand{ \c }{ \colon } \newcommand{ \ce }{ \mathrel{:=} } \newcommand{ \obj }{ \opname{obj} } \newcommand{ \hom }{ \opname{hom} } \newcommand{ \id }{ \mathrm{id} } \newcommand{ \ker }{ \opname{ker} } \newcommand{ \max }{ \opname{max} } \newcommand{ \C }{ \mathcal{C} } \newcommand{ \N }{ \mathbf{N} } $$

1.

We recall the definitions and prove two basic statements about finite presentation, that “finitely presented implies always finitely presented”1 and that it is a local property. The proofs in the setting of modules and in the setting of algebras are completely analogous, so we will abstract very slightly and address both at once.

2.

Notation.

(2.1)

All rings and algebras are commutative with unit.

(2.2)

For a ring $A$, let $\C_A$ denote either the category of $A$-modules or the category of $A$-algebras. All arrows indicate morphisms in $\C_A$.

(2.3)

Let $\amalg$ denote the coproduct in $\C_A$, i.e., direct sum $\oplus$ for $A$-modules or tensor product $\otimes_A$ for $A$-algebras.

(2.4)

For $C \in \obj(\C_A)$ and $n \in \N$ let $C^{\amalg n}$ denote the $n$-fold coproduct $C \amalg \cdots \amalg C$ and $C^n$ denote the $n$-fold product $C \times \cdots \times C$.

(2.5)

Let $F_A \in \obj(\C_A)$ denote the free object on one element, i.e., $F_A \iso$ $A$ for $A$-modules and $F_A \iso$ $A[t]$ for $A$-algebras. Thus for any $n \in \N$ there is an isomorphism $\hom_{\C_A}(F_A^{\amalg n},C) \iso$ $C^n$ natural in $C \in \obj(\C_A)$.

And now fix a ring $A$ for the remainder.

3.

Definition.

An object $C \in \obj(\C_A)$ is:

(3.1)

finitely generated if for some $n \in \N$ there is a surjection $F_A^{\amalg n} \to$ $M$;

(3.2)

finitely presented if for some $n \in \N$ there is a surjection $F_A^{\amalg n} \to$ $M$ with finitely generated kernel.

4.

Lemma.

Let $f \c C \to$ $D$ and $g \c D \to$ $E$ be morphisms in $\C_A$. Define $h \ce g \circ f$. Then if $f$ is surjective, the canonical morphism $\ker(h) \to$ $\ker(g)$ induced by $f$ is surjective. In particular, if $\ker(h)$ is finitely generated then so is $\ker(g)$.

4.1.

Proof.

Evident.

5.

Proposition.

Let $C \in \obj(\C_A)$ be finitely presented. If $\alpha \c D \to$ $C$ is a surjection with $D$ finitely generated, then $\ker(\alpha)$ is finitely generated.

5.1.

Proof.

By (4) we may assume $D \iso$ $F_A^{\amalg m}$. By definition we have a surjection $\beta \c F_A^{\amalg n} \to$ $C$ with $\ker(\beta)$ finitely generated. Consider the diagram

$$ \begin{xy} \xymatrix { & F_A^{\amalg m} \ar@{>>}[dr]^\alpha & \\ F_A^{\amalg m} \amalg F_A^{\amalg n} \ar@{>>}[ur]^{\id\,\amalg\,\xi} \ar@{>>}[dr]_{\eta\,\amalg\,\id} \ar@{>>}[rr]^-\gamma & & C, \\ & F_A^{\amalg n} \ar@{>>}[ur]_\beta & } \end{xy} $$

where $\xi \c F_A^{\amalg n} \to$ $F_A^{\amalg m}$ and $\eta \c F_A^{\amalg m} \to$ $F_A^{\amalg n}$ are defined such that $\alpha \circ \xi = \beta$ and $\beta \circ \eta = \alpha$—which exist since free objects are projective in $\C_A$—so that defining $\gamma \ce \alpha \amalg \beta$ makes the diagram commute. Now, if $t_1,\ldots,t_m$ are the canonical generators (in the appropriate sense according to $\C_A$) of $F_A^{\amalg m}$, then clearly $\ker(\gamma)$ is generated by $\ker(\beta)$ and $\{t_i - \eta(t_i)\}_{1 \le i \le m}$. Thus $\ker(\beta)$ being finitely generated implies $\ker(\gamma)$ is finitely generated, which implies $\ker(\alpha)$ is finitely generated by (4).

6.

Proposition.

Let $C \in \obj(\C_A)$.

(6.1)

If $C$ is finitely generated (resp. presented), then for any $f \in A$ the localisation $C_f$ is finitely generated (resp. presented) in $\C_{A_f}$.

(6.2)

If $f_1,\ldots,f_n \in A$ generate the unit ideal and $C_{f_i}$ is finitely generated (resp. presented) in $\C_{A_{f_i}}$ for each $i$, then $C$ is finitely generated (resp. presented).

6.3.

Proof.

We first address finite generation. Observe $(F_A)_f \iso$ $F_{A_f}$. Thus, since localisation is exact, hence commutes with finite products, a surjection $F_A^{\amalg n} \to$ $C$ localises to a surjection $F_{A_f}^{\amalg n} \to$ $C_f$. This proves (6.1). Now suppose for each $i$ that $C_{f_i}$ is generated (in the appropriate sense) by $\{a_{i,j}/f_i^{r_{i,j}}\}_{1 \le j \le m_i}$. Take $R \ge \max\{r_{i,j}\}$. Then for all $i$ we have that $f_i^Ra$ is generated by $\{a_{i,j}\}$ for any $a \in A$. Since $f_1,\ldots,f_n$ generate the unit ideal, so do $f_1^R,\ldots,f_n^R$ and hence there exist $c_i \in A$ such that $\sum c_if_i^R = 1$. It is then clear that $\{c_i\} \cup \{a_{i,j}\}$ generates $C$. This proves (6.2).

Now finite presentation. Since localisation is exact, hence commutes with kernels, (6.1) for finite presentation is immediate from (6.1) for finite generation. Now assume the hypothesis of (6.2). Since we have proved (6.2) for finite generation there is a surjection $\alpha \c F_A^{\amalg n} \to$ $C$. Then for each $i$,

$$ \alpha_{f_i} \c F_{A_{f_i}}^{\amalg n} \to C_{f_i} $$

is a surjection, so by (5) we know $\ker(\alpha_{f_i}) \iso$ $\ker(\alpha)_{f_i}$ is finitely generated. By (6.2) for finite generation this implies $\ker(\alpha)$ is finitely generated as desired.

7.

Ok that’s it! I mostly wanted to write this up because you can prove it in both settings at once, which I found fun. I thought a bit about axiomiatising the type of category one could prove this for, but I ended up wanting to assume a lot of stuff and it seemed like the exposition would get pretty ugly.

8.

Remark.

(6) is what one needs to prove when showing that a morphism of schemes being of locally of finite type or locally of finite presentation is local on the source.
